Ocean Environmental Monitoring System 🌊

This project is a hardware prototype designed to monitor ocean health and environmental conditions using an Arduino-based sensor platform. The system collects real-time data on temperature, humidity, distance, and environmental conditions, and displays results on an LCD while transmitting data through Bluetooth.

The goal is to create a low-cost, modular ocean monitoring platform that could be adapted for applications such as:

Coral reef health monitoring

Ocean pollution detection

Plastic debris detection

Environmental data collection for marine research

Features

📟 I2C LCD Display for live sensor readings

📡 HC-06 Bluetooth module for wireless data transmission

🌡 DHT11 Sensor for temperature and humidity monitoring

📏 Ultrasonic sensor for distance/object detection

⚙️ Servo control for mechanical movement or sampling systems

💡 LED indicators for system status alerts

Hardware Used

Arduino Mega 2560

HC-06 Bluetooth module

16x2 I2C LCD display

DHT11 temperature/humidity sensor

HC-SR04 ultrasonic sensor

Servo motor

LEDs and resistors

Applications

This prototype demonstrates how embedded systems and sensors can be used to monitor ocean environments, with potential applications in:

Coral reef monitoring

Water quality sensing

Autonomous ocean rovers

Environmental data collection
